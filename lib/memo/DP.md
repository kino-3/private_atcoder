# DP

## 問題

- 並べた 2N 個の要素のペアリングの作る手順の場合の数(同じ結果でも作る順番が違えば区別される)
- 制約: ペアリングがクロスしない
  - OK: AABB, ABBA
  - NG: ABAB

## 解法

- 区間DP: dp[l][n] = [l, l+2n) の部分の結果
- l とペアになる k は l+1, l+3, ..., l + 2i - 1, ..., l+2n-1
- 手順では [l, k], [k+1, r) の群のどちらかを選んでペアを1つつくることを繰り返すので, nCi
- dp[l][r] = Σ_k (dp[l+1][k] * dp[k+1][r] * nCi)
