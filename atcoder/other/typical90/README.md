## 002

- 1: n==20 なので全探索でよいことに気付く

解法は多分同じ。「(i & (1 << j)) = 0 というのは、i の j ビット目（2^j の位）が 0 であるための条件」に気付かなかったが, bitset の方が分かりやすい気がする。
