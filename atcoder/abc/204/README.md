## E

- t は整数なので, `t + C + [D/(1+t)] = C - 1 + [(t+1)+D/(1+t)]` として相加平均・相加平均からある時刻 T に頂点 u にいるときに, 隣接する頂点 v に行ける最短時間が分かる。よって, Dijkstra のコードを少し変えれば解けそうである。ということに気付くまでに結構時間がかかった。
- 道路が双方向であることに気付かずに WA の解消が遅れた
